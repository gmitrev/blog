This is an incomplete list of my personal projects created throughout the years.

Stonksfolio

Stonksfolio is a stock/ETF portfolio tracker that provides insights into the performance of your portfolios.

Supports most popular brokers with an optional CSV import for others.

Includes tools for benchmarking, rebalancing and measuring FX impact.

Are Na Bas

/бас/ - a bet between two parties, a wager

A place to remember bets and wagers between friends. Never again forget about that beer you are owed!

Nadir

/ˈneɪdɪə,ˈnadɪə/ - the lowest or most unsuccessful point in a situation.

Nadir is a blazing-fast exception tracker for Ruby applications that will be there for you when your code fails you.

Temporarily unavailable while moving to a new hosting provider.

Projects

This is an incomplete list of my personal projects created throughout the years.

Stonksfolio

Stonksfolio is a stock/ETF portfolio visualizer that provides insights into the performance of your portfolios.

It also has an increasing number of tools that help passive investors with their investments, such as an automatic contribution calculator.

Nadir

/ˈneɪdɪə,ˈnadɪə/ - the lowest or most unsuccessful point in a situation.

Nadir is a blazing-fast exception tracker for Ruby applications that will be there for you when your code fails you.

Offlife

Offlife is a collection of helpful applications for the daily life.

  • Budgeting and money management
  • Weight loss calulator and tracker
  • Progress pic visualizer
